Click the Github icon next to the title to see the code


The add trail card uses an input feild to collect the trail name, I cna then use the input to create new html elements to display the trails underneath. I can toggle the css to cross of the trail with a click eventlistener. The delete button runs a function to remove the created html element.

The get forecast card uses the Weatherbit API. Each button has an assigned value with the city name, the value is passed to the API using an axios get request. Then the api responds with the corresponding data in an object, the object is destructured and displayed with created html elements.
