SkiPak



Click the Github icon next to the title to see the code


The add trail card uses an input field to collect the trail name. Then, I use the input to create a new HTML element to display the trails underneath. I can toggle the CSS to cross off the trail with a click event listener. The delete button then runs a function to remove the created HTML element.

The forecast card uses a Weatherbit API. Each button has an assigned value with the city name. The value is passed to the API using an Axios GET request. Then, the API responds with the corresponding data in an object - the object is destructured and displayed with created HTML elements.
