SkiPak


SkiPak is a web application that allows the user to keep an aspirational list of backcountry trails, you can add a trail name, cross it off when skied, or delete the trail entirely. This app was built using HTML5, CSS3, and vanilla JavaScript. It also utilizes axios and the Weatherbit API.
